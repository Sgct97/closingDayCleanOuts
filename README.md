## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
Server will start at `http://localhost:4321`

### Build
```bash
npm run build
```

### Preview
```bash
npm run preview
```
